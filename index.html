Must use CSS Properties

Looking to learn CSS? Here are some of the CSS properties you'll use daily!

CSS Property Description Possible Values
background-color Sets the background color of an HTML element Any number of preset CSS colors, RGB combinations, or Hexadecimal codes
background-image Sets a background image for an HTML element Link to your image: url(".../media/image.png")
border Creates a border for your HTML element solid, dashes red, thick double blue
color Sets the color of your HTML element Any number of preset CSS Colors, RGB combinations, or Hexadecimal codes
display Specifies the display behavior of an html element none, inline, block, inline-bloc
font-family Specififies the font for HTML text elements Arial, Helvetica, "Bebas Neue"
font-size Specififies the font size HTML text elements 12px, 1.5em, small, 80%
font-weight Specififies the boldness HTML text elements 200, bold, normal
height Specififies the height of an HTML element 100px, 50%, 3em
margin Creates spacing around an HTML element 2em, 10px, 0
margin Creates spacing around an HTML element 2em, 10px, 0. Can also use margin-left, right, top, bottom
padding Creates spacing within an HTML element 2em, 10px, 0. Can also use padding-left, right, top, bottom
position Sets an HTML elements positioning within a webpage static, relative, absolute. Can use left, right, top, bottom properties
text-align Sets the alignment of HTML text elements center, left, right
text-transform Sets the capitalization of HTML text elements capitalize, uppercase, lowercase
width Sets the width of HTML elements 150px, 20em, 75%
